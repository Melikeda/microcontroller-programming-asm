# Microcontroller Programming ASM

Assembly language examples and practice codes developed during my 3rd year university Microcontroller Programming course.

---

## About This Repository

This repository contains various Assembly (ASM) programming examples created for laboratory studies, homework, and practice during the Microcontroller Programming course.

The purpose of these examples is to improve low-level programming knowledge and better understand how microcontrollers work internally.

---

## Topics Covered

- Assembly Language Fundamentals
- Register Operations
- Arithmetic Instructions
- Logical Instructions
- Loop Structures
- Branching Operations
- Memory Operations
- Input / Output Operations
- Basic Microcontroller Concepts
